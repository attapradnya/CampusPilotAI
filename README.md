# CampusPilot AI

> AI-powered academic assistant for university students.

## What is CampusPilot AI?

CampusPilot AI reads university emails, summarizes them using AI, extracts important information such as deadlines, meeting details, dress codes, venues, and links, and automatically syncs important events with Google Calendar.

## Features

- 📧 Smart email summaries
- 📅 Google Calendar integration
- ⏰ Deadline tracking
- 👔 Dress code detection
- 📍 Venue extraction
- 🔗 Link extraction
- ⭐ Priority detection
- 🤖 Powered by Google Gemini AI

## Status

🚧 Currently under development.
## Future Features

### AI Prompt Engine

For every incoming email:

- Summarize the email
- Identify the sender
- Extract deadlines
- Extract meeting date and time
- Extract venue
- Extract dress code
- Extract important links
- Identify action items
- Automatically create Google Calendar events
- Add reminders 30 minutes and 5 minutes before deadlines or meetings
